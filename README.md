# API
`Joao_Ramos_PB_TP1/`
Projeto REST API em Java SpringBoot.

# Frontend
`Joao_Ramos_PB_TP1_frontend/`
Projeto React que consome os endpoints da API.

## Rodando Localmente
1. Faça o download do repositorio 
2. Rode a API
    ```
    cd Joao_Ramos_PB_TP1/
    mvn spring-boot:run
    ```
3. Em um novo terminal, rode o app frontend
    ```
    cd Joao_Ramos_PB_TP1/
    npm install
    npm run
    ```
4. Acesse o portal atravéz de `http://localhost:3000`
